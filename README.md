#  Tela visual (HTML + CSS)

Este é um projeto simples desenvolvido com o objetivo de praticar **HTML** e **CSS**, focando na criação de uma **tela apenas visual**.

##  Sobre o projeto

A proposta foi construir apenas o **front-end** de uma tela, sem integração com back-end ou autenticação real.

O foco principal está em:

* Estruturação semântica com HTML
* Estilização moderna com CSS


##  Tecnologias utilizadas

* HTML5
* CSS3

##  Objetivo

Este projeto foi desenvolvido com fins de estudo para:

* Melhorar habilidades em layout
* Entender melhor posicionamento e organização de elementos
